Descripción:
Este proyecto MLOps utiliza FastAPI para crear una API que ofrece acceso a tres conjuntos de datos de Steam: user_reviews, user_items y steam_games. Incluye limpieza de datos, análisis exploratorio de steam_games y varios endpoints para acceder a la información. Destacamos un endpoint con un sistema de recomendación de juegos basado en similitud de coseno. Además, el despliegue se realiza con Render.

## Características principales

- Análisis de datos de Steam
- API con FastAPI
- Sistema de recomendación de juegos
- Despliegue con Render
